# AgriSense: IoT-Based Precision Agriculture System

![AgriSense](https://precision-agriculture.netlify.app/)

🌍 **Live Demo:** [https://precision-agriculture.netlify.app/](https://precision-agriculture.netlify.app/)

## Overview
AgriSense is a comprehensive local IoT system that monitors agricultural parameters (temperature, humidity, soil moisture), controls water pumps automatically based on weather-aware logic, and utilizes AI to provide actionable crop insights through a responsive web dashboard.

## Key Features
- **Live Dashboard Telemetry:** Displays critical farm health metrics.
- **Weather Integrated AI:** Provides actionable recommendations based on present sensors + future weather.
- **Smart Irrigation:** Eliminates water waste via rain-delay logic.
- **Tank UI & Alerts:** Visualizes the water tank level.
- **Dark Mode Interface:** Polished UI logic ensures readability.
